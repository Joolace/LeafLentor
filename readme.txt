Leaflet Map Widgets for Elementor provides three Elementor widgets to easily integrate interactive maps into your website.

== Description ==

LeafLentor is the perfect tool for adding beautiful, interactive maps to your WordPress website without touching a single line of code. This user-friendly plugin seamlessly integrates the powerful Leaflet mapping library with Elementor, the popular page builder.

Key Features:
- Drag-and-Drop Simplicity: Easily add a map to your Elementor-designed pages with the intuitive LeafLentor widget.
- Customization Options: Choose from various map providers, add markers with custom icons and popups, overlay additional data layers, and style the map to match your website's aesthetic.
- Flexibility and Power: Leaflet's extensive features give you complete control over your maps, from basic to advanced functionality.
- Responsive Design: Your maps will look stunning and function flawlessly on any device.
- Open Source and Free: LeafLentor is a community-driven project, free to use and modify.

Whether you're showcasing your business locations, highlighting points of interest, or creating custom map-based applications, LeafLentor empowers you to enrich your website with engaging and informative maps.

Get started today and bring your WordPress site to life with LeafLentor!

== Frequently Asked Questions ==

= What is LeafLentor? = 

LeafLentor is a WordPress plugin that seamlessly integrates the Leaflet mapping library with Elementor, a popular page builder. This allows you to easily create interactive maps and embed them directly into your Elementor-designed pages.

= Why should I use LeafLentor? =

Ease of Use: No coding is required! The intuitive interface makes it simple to add markers, custom icons, popups, and more.
Customization: Tailor the map's appearance to match your website's design.
Flexibility: Leaflet is a lightweight and powerful library, giving you a wide range of options for map features.
Elementor Integration: Works seamlessly within your existing Elementor workflow.

= What features does LeafLentor offer? =

Map Types: Choose from various map providers (OpenStreetMap, Mapbox, etc.).
Markers: Add markers with custom icons and informative popups.
Layers: Overlay additional information like heatmaps, GeoJSON data, or custom tiles.
Controls: Include zoom controls, layer switching, and other interactive elements.
Responsive Design: Your maps will look great on any device.

= How do I get started with LeafLentor? =

Install: Download and install the plugin from the WordPress repository or upload it manually.
Activate: Activate the plugin in your WordPress dashboard.
Create a Map: Use the LeafLentor widget within Elementor to design your map.
Customize: Adjust settings, add markers, and style your map to your liking.

= Is LeafLentor Widget free? =

Yes, LeafLentor is a free and open-source plugin.

= Where can I find support or report issues? =

You can find support and report issues on the plugin's GitHub page: https://github.com/Joolace/leaflet-elementor

= Can I contribute to LeafLentor? =

Absolutely! Contributions are welcome. Feel free to fork the repository on GitHub and submit pull requests.

Let me know if you'd like any adjustments or additions to these FAQs!
